## 分隔符

请补全JavaScript代码，要求返回参数数字的千分位分隔符字符串。

输入：
_comma(12300)
复制
输出：
'12,300'

法一，运用js
```js
    function _comma(number) {
        // 补全代码
        let arr = Math.abs(number).toString().split('').reverse();
        let i = 3;
        while (i<arr.length) {
            arr.splice(i, 0, ',')
            i += 4
        }
        let str = arr.reverse().join('')
        return number>0?str:'-'+str
    }

```
法二，运用正则表达式
```js
   function _comma(number) {
        // 补全代码
        return ('' + number).replace(/\d{1,3}(?=(\d{3})+$)/g, (val) => val+',')
    }
```
