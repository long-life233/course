# 简历
## 项目
名称：霆方氢汽小程序

介绍：该小程序的功能是为车主、消费者、运营人员提供服务。在此小程序上，车主可以选购房车、体验社区生活；消费者可以预定营地和房车；运营人员可以方便的管理房车营地，及时响应住客的服务需求。

人员周期：前端1个，首期3个月，迭代周期为1-2周

技术体系：uniApp、Vue3、Vuex、luch-request、Animate.css、Html、Css，JavaScript。

技术要点：瀑布流页面开发；luch-request请求的封装；用户授权及登录的Token认证；使用uniApp可发布到微信小程序和H5端；全局路由跳转的封装；交互效果和动画的制作；本地数据存储。
