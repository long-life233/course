# 云豹直播带货系统介绍

- 共有==五个端==，分别是：网页端、安卓APP端、IOS APP端、小程序端、公众号端（H5网页）

- 可以实现==数据互通==，用户在某一端注册账号，可在其他端直接登录使用

- 最新云豹直播系统源码大小在==800M==左右，不超过1G

- 使用==原生语言==进行开发（除公众号端受微信生态系统的限制，必须使用H5开发外），开发语言分别是：PHP（后台）、JAVA（安卓APP端）、OC（IOS APP端）、微信小程序原生语言。

- ==全球==服务器节点皆可搭建

- 各端可==独立==开播与观看。

- 免费==一年==售后和技术支持
